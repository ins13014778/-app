# MyApplication

这是一个基于巴法云开发，并结合 AI 进行二次开发改造的智能应用项目。

## 项目简介

本项目最初基于巴法云相关能力进行开发，在此基础上结合 AI 能力进行了进一步的功能扩展与改造，用于实现更智能化的应用体验。项目当前已经接入扣子工作流，可用于扩展智能问答、自动化处理和业务流程联动等能力。

## 项目组成

- `entry/`：HarmonyOS 应用前端主要代码
- `api/`：项目后端 API 目录，包含接口、配置、数据库脚本、模型及相关服务代码
- `.idea/`、`.hvigor/`、`build/`、`oh_modules/`：当前本地工程相关文件也已一并纳入仓库

## 功能说明

- 基于巴法云能力进行项目开发
- 使用 AI 对项目进行了二次开发与功能改造
- 已接入扣子工作流，用于扩展智能化业务流程能力
- 已包含 `api/` 后端接口目录，便于前后端联调与整体部署

## API 说明

`api/` 目录已上传到仓库，当前包含以下内容：

- 配置文件与数据库连接配置
- 用户、资料、消息等相关接口
- 设备状态、指令、传感器数据相关接口
- 数据库初始化脚本与数据文件
- MQTT 相关处理脚本

## 使用说明

你可以基于本项目继续进行个性化开发，并结合自己的业务场景进一步扩展。
