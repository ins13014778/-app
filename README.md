# MyApplication

这是一个基于巴法云开发、由原创作者在 AI 辅助下完成设计与开发的智能应用项目。

## 项目简介

本项目由作者原创开发，AI 在开发过程中主要作为辅助工具参与方案整理、代码生成参考、功能改造和开发效率提升。项目核心需求、功能设计、业务整合与整体实现方向均由作者本人主导完成。

项目当前已经接入扣子工作流，可用于扩展智能问答、自动化处理和业务流程联动等能力。

## 项目特点

- 基于巴法云相关能力进行开发
- 由原创作者主导设计与实现
- 使用 AI 辅助完成部分开发与二次改造工作
- 已接入扣子工作流，扩展智能化业务流程能力
- 已包含 `api/` 后端接口目录，便于前后端联调与整体部署

## 项目结构

- `entry/`：HarmonyOS 应用前端主要代码
- `api/`：项目后端 API 目录，包含接口、配置、数据库脚本、模型及相关服务代码
- `.idea/`、`.hvigor/`、`build/`、`oh_modules/`：当前本地工程相关文件也已纳入仓库

## API 说明

`api/` 目录当前包含以下内容：

- 配置文件与数据库连接配置
- 用户、资料、消息等相关接口
- 设备状态、指令、传感器数据相关接口
- 数据库初始化脚本与数据文件
- MQTT 相关处理脚本
